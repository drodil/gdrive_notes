# GUI commands:

* `j` / `k`: Move up down
* `:q`: Quit
* `:q!`: Quit without saving
* `:qw`: Save and quit
* `a`: Add new note
* `D`: Delete selected note
* `e`: Edit selected note
* `Enter`: Show note details / content
* `G`: Go to bottom of the list
* `:h`: Print help
* `:a <note>`: Quick add note
